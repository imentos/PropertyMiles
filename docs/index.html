How It Works

1. Install & Grant Permission

Download the app and allow "Always" location access. This enables automatic trip detection.

2. Drive Normally

That's it! The app automatically detects trips when you drive. No need to open the app.

3. Review & Tag

After trips are recorded, tap to add purpose, property, and notes. Takes seconds.

4. Export for Taxes

At tax time (or anytime), export a complete CSV report with one tap. Done.
